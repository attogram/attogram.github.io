Attogram Framework

The Attogram Framework provides developers a PHP skeleton starter site with content modules, file-based URL routing, IP-protected backend, user system, integrated SQLite database with web admin, Markdown parser, jQuery and Bootstrap. After that, Attogram tries to stay out of your way while you do your thing!
